# 👨🏻‍💻 Pedro Wilson

**`Desenvolvedor Front-End`**

Bacharel em Engenharia de Software (2020 - 2024) e Tecnólogo em Design Gráfico (2018 - 2019), sou um Desenvolvedor Front-End com sólida base em design visual. Ao longo da minha trajetória, tenho me dedicado à criação de interfaces web e mobile, unindo estética e funcionalidade. Iniciei minha carreira desenvolvendo habilidades em design visual utilizando o Photoshop, o que me proporcionou uma base sólida para transformar conceitos criativos em interfaces intuitivas e eficientes por meio da programação front-end.
